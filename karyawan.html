Undang Karyawan Baru

Cara menambah karyawan baru: Undang karyawan melalui Supabase. Mereka akan mendapat email berisi link untuk set password. Setelah login pertama, profil mereka muncul otomatis di daftar ini dan Anda bisa melengkapi datanya.

Fitur kirim invite langsung dari aplikasi membutuhkan konfigurasi tambahan. Untuk saat ini, undang karyawan melalui:
Supabase Dashboard → Authentication → Users → Invite user
